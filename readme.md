# Browsersverige.se

## Vad surfar svenskarna med?

Simple site showing which browsers the Swedes use to browse the Internet.

![](https://browsersverige.se/images/og.jpg)

Hosted on Zeit Now

Run

```
yarn start
```

Get browser statistics data

```
yarn getdata
```

Deploy

```
yarn deploy
```

Release (Get data and deploy)

```
yarn release
```
